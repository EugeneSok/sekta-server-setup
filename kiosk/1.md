# Logo splash screen   
**Покрокова інструкція**  
```
# 1. Встановити plymouth
sudo apt install plymouth plymouth-themes

# 2. Створити папку теми
sudo mkdir /usr/share/plymouth/themes/mylogo

# 3. Скопіювати логотип
sudo cp /home/pi/omega.png /usr/share/plymouth/themes/mylogo/logo.png

# 4. Створити файл теми
sudo nano /usr/share/plymouth/themes/mylogo/mylogo.plymouth

```
Вставити:  
```
[Plymouth Theme]
Name=MyLogo
Description=My custom logo
ModuleName=script

[script]
ImageDir=/usr/share/plymouth/themes/mylogo
ScriptFile=/usr/share/plymouth/themes/mylogo/mylogo.script


```
# 5. Створити скрипт  
```
sudo nano /usr/share/plymouth/themes/mylogo/mylogo.script

```
Вставити:  
```
logo = Image("logo.png");
screen_width = Window.GetWidth();
screen_height = Window.GetHeight();

scale = Math.Min(
    screen_width  / logo.GetWidth(),
    screen_height / logo.GetHeight()
) * 0.4;

scaled = logo.Scale(
    logo.GetWidth()  * scale,
    logo.GetHeight() * scale
);

sprite = Sprite(scaled);
sprite.SetX(screen_width  / 2 - scaled.GetWidth()  / 2);
sprite.SetY(screen_height / 2 - scaled.GetHeight() / 2);

```
# 6. Активувати тему  
```
sudo plymouth-set-default-theme -R mylogo

```
# 7. Прибрати текст завантаження в GRUB  
```
sudo sed -i 's/GRUB_CMDLINE_LINUX_DEFAULT=.*/GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"/' /etc/default/


```
```
sudo update-grub

```
**Перевірка без перезавантаження**  
```
sudo plymouthd --no-daemon & sudo plymouth --show-splash && sleep 5 && sudo plymouth quit

```
Потім просто перезавантаж:  
```
sudo reboot 

```
