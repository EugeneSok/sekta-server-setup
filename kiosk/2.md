## ✅ Надійний спосіб: користувацький systemd-юніт (рекомендовано)  
Працює після входу користувача в графічну сесію (GNOME/KDE/XFCE), не треба вручну виставляти DISPLAY.  
1. Створи юніт:  
```

mkdir -p ~/.config/systemd/user
nano ~/.config/systemd/user/firefox-kiosk.service

```
2. Встав:  
```

[Unit]
Description=Firefox Kiosk (user)
After=graphical-session.target
PartOf=graphical-session.target

[Service]
Type=simple

```
```
ExecStart=/usr/bin/chromium --kiosk https://10.10.10.50
Restart=on-failure

```
```
RestartSec=2

[Install]
WantedBy=graphical-session.target

```
3. Увімкни:  
```

systemctl --user daemon-reload
systemctl --user enable --now firefox-kiosk.service

```
🔹 Якщо треба, щоб воно стартувало **навіть після автологіну без відкриття терміналу**, увімкни “user lingering”:  
```

sudo loginctl enable-linger "$USER"

```
